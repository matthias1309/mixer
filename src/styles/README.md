# Styles Directory

Global and component-specific styling.

## Structure

- **globals.css** - Global styles and CSS variables
- **variables.css** - Design tokens (colors, spacing, fonts)
- **[component].module.css** - CSS modules for component-specific styles

## Conventions

- Use CSS modules for component scoping to avoid conflicts
- Define color and spacing tokens in variables.css
- Global styles only for resets and utilities
- Follow BEM naming for global classes if needed
