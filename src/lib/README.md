# Lib Directory

Utility functions, helpers, and business logic.

## Structure

- **constants.ts** - Application-wide constants
- **db.ts** - Database connection and utilities
- **auth.ts** - Authentication helpers (JWT, hashing, verification)
- **validation.ts** - Input validation functions
- **errors.ts** - Custom error classes
- **[feature].ts** - Feature-specific utilities

## Conventions

- Pure functions where possible
- Well-documented, especially for complex logic
- Unit tested
- No side effects unless explicitly needed
