# App Directory

Next.js app directory containing pages, layouts, and route handlers using the App Router pattern.

## Structure

- **layout.tsx** - Root layout component
- **page.tsx** - Home page
- **(auth)/** - Authentication-related routes
- **(recipes)/** - Recipe management routes
- **(filters)/** - Filter/search routes

## Usage

Each subdirectory represents a route segment. Files like `page.tsx` define the UI for that route.
