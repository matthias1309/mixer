## Database Schema

No schema changes in this release. All tables from v1.0.0 remain:
- `users` — User accounts
- `recipes` — User-created recipes
- `ingredients` — Recipe ingredients
- `nutrition_ingredients` — Master ingredient database (29+ items)
- `recipe_nutrients` — Pre-calculated nutrition data
- `user_cycles` — Menstrual cycle tracking
- `ingredients_master` — Ingredient master database

## Support & Rollback

### View Logs
```bash
ssh dockerhome 'cd /opt/containers/apps/mixer && docker compose -f docker-compose.production.yml logs -f mixer-app'
```

### Stop Services
```bash
ssh dockerhome 'cd /opt/containers/apps/mixer && docker compose -f docker-compose.production.yml down'
```

### Rollback (if needed)
```bash
# On PI: restore from backup or redeploy v1.0.0
git checkout v1.0.0
./scripts/deploy-pi.sh --db-password "..." --jwt-secret "..."
```
